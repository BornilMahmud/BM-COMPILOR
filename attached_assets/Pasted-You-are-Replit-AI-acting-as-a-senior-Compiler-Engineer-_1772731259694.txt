========================
2) MINILANG LANGUAGE SPEC
========================
MiniLang is C-like with blocks:

- Types: int, float, bool, string
- Literals:
  int: 123
  float: 1.23
  bool: true / false
  string: "hello"
- Statements end with ';'
- Statements:
  - declaration:
      int a;
      float b;
      string s;
      bool ok;
  - declaration with initialization:
      int a = 10;
      float f = 2.5;
      string s = "hi";
      bool ok = true;
  - assignment:
      a = a + 1;
  - print:
      print(a);
      print("text");
      print(a + b);
  - if/else:
      if (a > 0) { print(a); } else { print(0); }
  - while:
      while (a > 0) { a = a - 1; }
- Expressions:
  - arithmetic: + - * / (int/float)
  - comparison: == != < <= > >=  (results bool)
  - parentheses supported
- Comments:
  // line comment
  /* block comment */
- Scoping: variables declared inside { } are block-scoped.

TYPE RULES (semantic analysis must enforce):
- numeric ops only on int/float
- implicit int -> float allowed in expressions and assignments to float
- string:
  - allowed: string + string (concatenate)
  - disallow: string * int etc
- conditions in if/while must be bool (ONLY bool; no implicit numeric-to-bool)
- cannot use undefined variables
- cannot redeclare in same scope
- print() accepts any type
